## 介绍

**ToolKits** 为一款工具集软件，目前包含以下工具：

### 提取文件夹文件内容

选择文件夹之后，输入需要选择的文件的条件，然后读取根据条件选择的文件内容。

#### 输入内容

1. 包含条件
    - 文件夹名前缀
    - 文件加夹后缀
    - 部分文件夹名
    - 文件夹全名
    - 文件名前缀
    - 文件名后缀
    - 部分文件名
    - 文件全名

2. 排除条件
    - 文件夹名前缀
    - 文件加夹后缀
    - 部分文件夹名
    - 文件夹全名
    - 文件名前缀
    - 文件名后缀
    - 部分文件名
    - 文件全名

#### 输出内容

- 文件夹目录结构
- 已读取的文件名列表
- 不能读取的文件名列表
- 读取到的文件内容
